Cette appplication permet de contrôler diverses expérimentations relatives à la mise en oœuvre de capteurs de lumière pour :

  • détecter la présence d'un objet sans contact
  • distinguer un objet blanc d'un objet noir
  • distinguer des objets de couleurs différentes

Informations techniques :

  • serveur développé en Python sur la base du framework Tornado
  • programmation client :
    • jQuery
    • plugin jQuery validation
  • habillé par Bootstrap et Bootswatch Slate
